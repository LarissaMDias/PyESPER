# PyESPER

This is a preliminary version of Python translation of the neural network implementation of Empirical Seawater Property Estimation Routines, or ESPERs, written by Brendan Carter (https://github.com/BRCScienceProducts/ESPER/tree/main). This package is not meant to be used professionally until the validation and assessment of the code is complete. Please note the following:

1. Only DIC and TA estimates are available (DesiredVariables).
2. LIR version has not yet been added to the package.

If you have any questions, please feel free to email me at lmdias@uw.edu.
